# PORTABLE-SPECTROMETER-USING-LDR-SENSOR-ESP32-GOOGLE-SHEET-BLYNK
This portable ESP32 spectrometer analyzes water quality using LDR and servo-controlled diffraction. It detects contaminants by wavelength, displays results on LCD, logs to Google Sheets, and transmits real-time data via Blynk (V0-V3) for remote monitoring. Combines hardware sensing with cloud/mobile visualization
